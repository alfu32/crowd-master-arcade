# Privacy Policy

Last updated: July 2, 2026

Crowd Master Arcade is an offline desktop application. It does not require an account and does not include telemetry by default.

## What data Crowd Master Arcade collects

- None by default.

## What data Crowd Master Arcade stores locally

- Project files you create or open.
- Application settings and preferences stored on your device.
- Optional plugin data stored by third-party plugins.

## Network access

- Crowd Master Arcade does not send data to external servers by default.
- Some features (such as plugins you install) may access the network. Those plugins are responsible for their own privacy practices.

## Third-party plugins

- Plugins are separate software components. Their data collection and network behavior may differ.
- Review the plugin’s documentation before installing.

## Changes to this policy

- This policy may be updated as the application evolves.
- Material changes will be documented in release notes.
